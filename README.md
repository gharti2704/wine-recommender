# wine-recommender

## Tech stack
1. Node
2. Express
3. React

## Project setup
1. npm install
2. Create .env file in the root directory and add
    1. API_KEY=your api key
    2. API_URL=https://api.spoonacular.com

## Running the project 
1. Server
    1. npm start
2. Client
    1. npm run react-dev
